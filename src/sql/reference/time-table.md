# Tabla de tiempo

La subconsulta predefinida **TIME_TABLE** permite obtener algunos de los campos habituales de una tabla de tiempo.

<div class="mt-1 mb-2 row">
  <div class="col-lg-12">

``` sql
select *
from time_table
where 
   year between 2000 and year(getdate())
```
